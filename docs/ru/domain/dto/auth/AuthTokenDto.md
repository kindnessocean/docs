# AuthToken

## Description
| FieldName | Type   | Description                     |
|-----------|--------|---------------------------------|
| token     | String | JWT токен                       |
| validity  | Long   | Валидность токена в миллисекунд |

## Example
```
{
    "token" : "1234567890qwertyuiopsdfghjklzxcvbnm",
    "validity" : 10000000
}
```
